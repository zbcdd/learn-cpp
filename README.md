# learn-cpp

C++ 学习仓库，用于阅读经典书籍、记录笔记、编写练习代码。

## 目录结构

```
references/   — 学习资料（PDF、书籍等）
notes/        — 读书笔记，按书籍分子目录
code/         — 练习代码、demo 示例
```

## 阅读计划

- [ ] *Effective Modern C++* — Scott Meyers (`notes/effective_modern_cpp/`)
